VRMOTION VRPLAYER - рекомендации по форматам

Проигрывание:
- Рекомендуемый формат видео: MP4 H.264/AAC.
- Для VR/360 потоков хорошо подходят MP4, MOV и M3U8.
- Для локальных файлов также поддерживаются MKV, WebM, JPG, PNG, WebP, AVIF.
- Если автоопределение режима ошиблось, выбирайте режим вручную: Auto, Flat 2D, 360, 360 3D SBS, VR180.

Запись REC:
- Рекомендуемый формат записи: MP4.
- Самый стабильный промежуточный формат для Electron: WebM.
- В PRO версии встроен FFmpeg, поэтому MP4, MOV, MKV, FLV и WMV создаются через конвертацию из стабильной промежуточной записи.
- Для максимальной совместимости используйте MP4.
- Для дальнейшего монтажа можно использовать MOV или MKV.
- WMV и FLV оставлены для совместимости со старыми системами и плеерами.

Качество REC:
- Рекомендуемый битрейт для 4K: 50000 кбит/с.
- Для меньшего размера файла можно ставить 30000-40000 кбит/с.
- Для максимального качества можно ставить 60000-100000 кбит/с, но файлы будут тяжелее.
- Частота записи ориентирована на 50 кадров/с.

PRO / LITE:
- PRO: есть REC, выбор формата записи, битрейт, чувствительность мыши и встроенный FFmpeg.
- LITE: без REC и без FFmpeg, но проигрывание и основные режимы сохранены.
