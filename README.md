# VRMOTION VRPLAYER LITE

Electron desktop build of VRMOTION VRPLAYER LITE.

## Mac build on GitHub

1. Open the repository on GitHub.
2. Go to `Actions`.
3. Select `Build Mac`.
4. Press `Run workflow`.
5. Download artifacts:
   - `VRMOTION-VRPLAYER-LITE-1.05-mac-x64`
   - `VRMOTION-VRPLAYER-LITE-1.05-mac-arm64`

The current workflow creates unsigned test builds. macOS may show a security warning on first launch.

For a public release without Gatekeeper warnings, add Apple Developer signing and notarization secrets.
